Electronic and vibrational polarizability tensors
-------------------------------------------------

The :class:`~mybigdft.workflows.poltensor.PolTensor` class allows to compute
the electronic polarizability tensor of a given system.

The :class:`~mybigdft.workflows.vibpoltensor.VibPolTensor` class allows to
compute the vibrational polarizability tensor of a given system.
